Methodology

How I built a running companion.

Swift SwiftUI Firebase Auth Cloud Firestore MapKit Core Location Xcode

1. Research & Problem Definition

Researched running injury statistics and identified that most injuries stem from progressing too quickly and found that existing apps lack personalized guidance for beginners. I can testify for both from my own experience.

2. Architecture & Data Model Design

Designed the app using Model View ViewModel architecture for clean separation of concerns. Used Firestore database for users, runs, and check-ins.

3. Authentication & User Profiles

Implemented Firebase Authentication for secure email/password login. Built an onboarding survey to capture experience level, goals, and running history using SwiftUI.

4. Run Tracking Implementation

Built GPS-based run tracking using Core Location and MapKit. Implemented real-time pace calculation, distance tracking, and route visualization with pause/resume functionality.

5. Recommendation Engine Development

Created a readiness-based algorithm in Swift that computes a score (0-100) based on recent run difficulty, pain levels, rest days, soreness, and sleep quality. The engine uses this score to suggest appropriate workouts.

6. Progressive Overload Logic

Implemented safe progression rules such as maximum 10% weekly volume increase along with goal-aware distance targets and automatic rest recommendations when pain or fatigue thresholds are exceeded.

7. UI/UX Design & Testing

Designed a clean SwiftUI interface with beginner-friendly messaging which uses relative pace analysis specific to the user.

Results & Features

What RunRight delivers for beginner runners.

Smart Recommendations

Personalized daily workout suggestions based on your unique data.

  • Requires 3 runs to build baseline
  • Considers pain, soreness, and sleep
  • Goal-aware progression (5K, 10K, etc.)
  • Automatic rest day suggestions

Pain & Recovery Tracking

Monitor your body's signals to prevent overtraining.

  • Post-run pain level (0-10 scale)
  • Specific pain area tracking
  • Daily soreness check-ins
  • Sleep quality consideration

Live Run Tracking

Real-time metrics during your runs.

  • GPS route tracking with MapKit
  • Live pace display
  • Distance and duration
  • Pause/resume support

Beginner-Friendly Design

Encouraging feedback that builds confidence.

  • Relative pace analysis (vs. YOUR average)
  • Supportive, encouraging messages
  • Educational explanations
  • Progressive goal tracking
